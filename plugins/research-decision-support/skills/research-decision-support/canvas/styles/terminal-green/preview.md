# Terminal Green — GitHub-dark terminal session, ANSI accents

## Palette

| page / surface | #0d1117 / #161b22 | #f6f8fa / #ffffff |

## Typography

Monospace only (13px, 1.5); hierarchy from weight and hue, never typeface changes.

## Signature moves

- Prompt line with blinking cursor heads each view; window-chrome frame around the whole.
- Content as aligned listing rows; bracketed argument chips; lowercase status footer.
- tabbed-gallery: numbered command chips switch pages; cards compress to listing rows.

## Fit

Best for: dev-facing demos, session walkthroughs, hacker-audience shareouts.
Avoid for: print handouts, non-technical stakeholder readouts.
