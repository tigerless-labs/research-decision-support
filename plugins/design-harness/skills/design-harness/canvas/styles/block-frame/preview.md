## Typography

Inter 900 uppercase display, tight tracking; wide-tracked grotesk label voice; 14px body.

## Signature moves

- 3px ink borders + 6px hard offset shadows, zero blur, square corners everywhere.
- Pastel fills cycle across surfaces; exactly one tilted element per view.
- tabbed-gallery: bordered pastel blocks as tabs (active pops with shadow + tilt).

## Fit

Best for: launch decks, agency-grade reviews, confident contemporary readouts.
Avoid for: quiet institutional restraint, regulated formal documents.
