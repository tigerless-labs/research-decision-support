# [论文] Do Self-Evolving Agents Forget? Capability Degradation and Preservation

**(authors not captured)** — [arXiv:2605.09315](https://arxiv.org/abs/2605.09315) · 2026-05 · no public code.

Documents **non-monotonic self-evolution**: self-improvement can degrade previously-held capability, and preservation must be designed in.

**Relevance to autoharness:** the case for **rollback + regression-gating as first-class** — net improvement on new tasks can hide regressions on old ones, so every change must be reversible and checked against prior behavior.
