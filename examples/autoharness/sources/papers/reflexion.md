# [论文] Reflexion: Language Agents with Verbal Reinforcement Learning

**Shinn, Cassano, Gopinath, Narasimhan, Yao** — [arXiv:2303.11366](https://arxiv.org/abs/2303.11366) · 2023.

Agents improve by converting outcome feedback into **verbal self-reflection** stored in an episodic buffer — reinforcement in language, not weights.

**Relevance to autoharness:** the canonical "reflect-then-edit" backward pass that the whole symbolic-learning line generalizes — `experience → reflection → symbolic artifact` (vs. `experience → gradient → weights`).
