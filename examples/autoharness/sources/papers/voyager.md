# [论文] Voyager: An Open-Ended Embodied Agent with Large Language Models

**Wang, Xie, Jiang et al. (NVIDIA / Caltech / others)** — [arXiv:2305.16291](https://arxiv.org/abs/2305.16291) · 2023.

The foundational **executable skill library**: an LLM agent in Minecraft that writes, stores, and reuses code skills, growing a self-extending library through environment feedback.

**Relevance to autoharness:** the origin point of "skill = reusable code artifact grown from experience" — the lineage the [symbolic-learning thesis](../blogs/symbolic-learning-renaissance.md) traces forward to AutoHarness and SkillOpt.
