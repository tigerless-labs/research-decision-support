# [博客] Learning Beyond Gradients (Heuristic Learning)

**Jiayi Weng** — referenced via the [symbolic-learning essay](symbolic-learning-renaissance.md); no stable URL captured.

Argues for moving the RL loop **from weight-space to software-space** — learning code / policy / state-detectors / tests / memory instead of weights ("Heuristic Learning").

**Relevance to autoharness:** the clearest statement of the mechanism autoharness implements — optimize the software artifacts around a frozen model, not the model. One of the three works the symbolic-learning thesis is built on.
