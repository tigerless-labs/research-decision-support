# Tabbed gallery template: evidence cards / idea cards / system diagrams on three separate pages

One option in the canvas template set (alongside
[single canvas, three clusters](single-canvas-clustered-graph.md); the human picks a style when
setting the target): one page per layer -- evidence cards and idea cards each get a gallery
page (tag-chip filtering + search; the card face shows only title + summary + a corner tag,
click to open the full text and its links), and a system-diagram page carries the flow and
architecture diagrams. The current artifact is a live instance of this template. Trade-off
against the infinite canvas: pagination wins on low barrier to entry and shareable static
pages; the canvas wins on cross-layer single-screen view and zoom semantics (see the
[creation canvas](../creation-canvas.md)).
