# Ideas and output are fully decoupled — no sync either way, only a far-behind reminder

Adjudicated 2026-07-16, superseding sync-on-command:

- **No sync in either direction.** Idea changes carry no obligation toward output, and a
  human edit to output stays in output — nothing is re-derived, nothing is
  back-transcribed. Divergence between the layers remains information, not an error.
- **The layers reconnect only when the human orders an assembly** — re-derivation is a
  human-initiated generation act, not a maintained relationship.
- **One duty across the gap**: when output has fallen far behind the ideas — many
  judgments landed since the last assembly — the agent may say so once ("output is N
  ideas behind"), never nagging.

Supersedes [sync on command](archive/sync-on-command-layers-independent.md) (archived on
the human's call, 2026-07-16), whose automatic back-transcription half also retires; the
back-calibration sentence of
