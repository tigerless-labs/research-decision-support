# [Method] PrOACT -- the textbook standard decision process

Hammond, Keeney & Raiffa, "Smart Choices" (Harvard, 1999). Backing: (3) each component is backed
by decision-analysis literature, the full process has no controlled validation.

Core logic: Problem -> Objectives -> Alternatives -> Consequences -> Trade-offs. Two key points:
audit the problem first -- problems are often framed wrong, and solving the right problem beats
solving the problem well; work through trade-offs with the **even-swaps method**, eliminating
dimensions step by step until what remains is the answer.

Boundaries: designed for major one-shot choices, heavy process; clarifying objectives depends on
the decision-maker's honesty.

**Relation to this project**: the textbook prototype of the decision card's expanded state --
problem + objectives (drivers) + option x consequence matrix + trade-offs, isomorphic to the
existing decision worksheet; filling the consequence table goes to the agent, problem definition
and trade-offs go to the human.
