# [Method] ACH, Analysis of Competing Hypotheses -- matrix adjudication and forced disconfirmation

Richards Heuer, "Psychology of Intelligence Analysis" (CIA, 1999). Standard method in the US
intelligence community.

Core logic: first enumerate all hypotheses exhaustively, then build a **hypothesis x evidence
matrix** and evaluate it cell by cell. The core insight: people naturally collect supporting
evidence (confirmation bias), so the methodology must **force unfavorable evidence into view** --
evaluation focuses on which piece of evidence *rules out* which hypothesis, not which supports
which. The least-refuted hypothesis wins.

Boundaries: coming up with the hypotheses is left to humans (the method only demands
exhaustiveness, it provides no generation procedure); in the intelligence community the
matrix-evaluation labor falls on analysts and is far from cheap.

**Relation to this project**: the origin of the decision card's core view (option x evidence
matrix); "when agents disagree, the disagreement must go on the panel -- no picking sides" shares
its roots with forced disconfirmation. The matrix-filling labor moves to the agent, eliminating
the human cost.
