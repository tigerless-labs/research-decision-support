# [产品] Connected Papers — 单种子相似图

connectedpapers.com · 免费 5 图/月，premium ~$6/mo。

核心逻辑：给一篇种子论文，按引用相似度长出一张无序图，快速发现开山作与综述。
最快上手的"领域一览"，但图是只读的：无标注、无判断、无后续工序。

**与 loom 的关系**：属"发现"环节，在 loom 收集来源之前；map 页视觉上易被类比，但语义
不同（相似度 vs 人工建立的溯源链）。来源：[三工具对比](https://effortlessacademic.com/litmaps-vs-researchrabbit-vs-connected-papers-the-best-literature-review-tool-in-2025/)。
