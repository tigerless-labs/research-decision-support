# [产品] ResearchRabbit — 迭代式引文追踪

researchrabbitapp.com · 免费为主，2025 改版后增付费层，与 Litmaps 结成伙伴。

核心逻辑：从 collection 出发迭代扩展引文网络，可"跳回"上一步检索路径；支持笔记、
协作、与 Zotero 双向同步。是"发现 + 收藏"工作流里体验最顺的一个。

边界：笔记是自由文本，不结构化；没有方向/判断/设计的后续层。

**与 loom 的关系**：上游发现工具，产出可以喂进 loom 的 sources 层。来源：
[2025 改版评测](https://aarontay.substack.com/p/researchrabbits-2025-revamp-iterative)。
