# [产品] Litmaps — 坐标化文献地图

litmaps.com · 免费层 + 订阅。

核心逻辑：多种子起图，X 轴=年份、Y 轴=被引数、点大小=引用量，可换轴自定义；
Monitor 每周推送新相关论文；有共同作者检索。三个图工具里可视化/过滤最强。

边界：同 Connected Papers——图服务于"发现"，不承载读后的判断与综合。

**与 loom 的关系**：上游发现；其"坐标轴可换"的思路对 map 页未来的排序维度有参考价值。
来源：[HKUST 图工具对比](https://libguides.hkust.edu.hk/citation-chaining/citation-mapping-tools-comparison)。
