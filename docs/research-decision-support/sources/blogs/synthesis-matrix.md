# [方法] Synthesis Matrix — 高校写作中心的综合矩阵惯例

多所大学图书馆/写作中心的标准教法（USU、JHU、Williams 等 LibGuides）。

核心逻辑：行=概念/主题、列=来源的表格，**边读边填**而非读完再补；格子的空缺直接
可视化"哪个论点证据不足"，矩阵里的不一致就是综述的写作线索。惯例载体是 Excel/Word
模板，纯手工。

**与 本项目 的关系**：compare 页的方法论祖先——本项目 把"边读边填"自动化（方向归属由
synthesis 卡链接推导 + 浏览器草稿），把"格子"升级为带溯源的判断。来源：
[USU 指南](https://libguides.usu.edu/c.php?g=648048&p=10935720)、
[JHU 指南](https://guides.library.jhu.edu/lit-review/synthesize)。
