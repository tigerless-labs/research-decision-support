# [Method] Synthesis Matrix -- the synthesis-matrix convention of university writing centers

The standard teaching of many university libraries/writing centers (USU, JHU, Williams, and other
LibGuides).

Core logic: a table with rows = concepts/themes and columns = sources, **filled while reading**
rather than backfilled after reading; empty cells directly visualize "which argument lacks
evidence", and inconsistencies inside the matrix become the writing threads of the review. The
convention lives in Excel/Word templates, fully manual.

**Relevance to this project**: the methodological ancestor of the compare page -- this project
automates "fill while reading" (direction membership derived from synthesis-card links + a browser
draft) and upgrades the "cell" into a judgment with provenance. Sources:
[USU guide](https://libguides.usu.edu/c.php?g=648048&p=10935720),
[JHU guide](https://guides.library.jhu.edu/lit-review/synthesize).
