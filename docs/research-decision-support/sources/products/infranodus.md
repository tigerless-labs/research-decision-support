# [Product] InfraNodus -- automatic text-network graphs and gap detection

infranodus.com · standalone product + Obsidian plugin, paid accounts.

Core logic: automatically builds a concept network from text/notes -- no manual linking; ranks
ideas by influence via betweenness centrality, detects "structural gaps" in the network, then uses
built-in AI to generate bridge-filling new ideas and research questions.

Boundary: stops at association discovery and divergence -- the output is graphs and new ideas,
with no option comparison, no evidence grading, no adjudication trace; a link is not a decision.

**Relevance to this project**: the nearest competitor and human-factors reference for the creation
canvas's "auto-linking" experience; also proof of the open slot -- auto-linking is already
commoditized, and "distilling links into decisions afterwards" is done by no one.
Source: [InfraNodus Obsidian plugin](https://infranodus.com/obsidian-plugin).
