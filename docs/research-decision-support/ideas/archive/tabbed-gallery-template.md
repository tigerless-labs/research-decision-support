# 分页画廊模板：证据卡 / 想法卡 / 系统图 三页分层

画板模板集的一款（与[单画布三团](single-canvas-clustered-graph.md)并列，定
target 时由人选风格）：每层一个页面——证据卡、想法卡各一页画廊（tag 筹码过滤 +
搜索，卡面只有标题+摘要+角标 tag，点开看全文与链路），系统图一页承载流程图与
架构图。当前 artifact 即此模板的活实例。与无限画布的取舍：分页胜在低门槛、
可分享静态页；画布胜在跨层一屏与缩放语义（见
[创作画板](../creation-canvas.md)）。
