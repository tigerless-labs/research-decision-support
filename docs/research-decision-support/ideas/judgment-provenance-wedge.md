# 差异化楔子：把"我的判断"作为一等、可溯源、通向设计的对象

竞品四簇（见 [sources 索引](../sources/index.md)竞品组）分别停在发现、提取、
筛选、存储；没有一家让判断带出处流向设计层。本项目 不与上游竞争（Zotero/发现图谱的产出
可直接喂进 sources 层），只做它们全都缺的下半程：判断 → idea → design → decision，
全程溯源。这也是通用性的来源——任何"读材料→做决定"的场景（选型、尽调、竞品分析）
都适用，不限学术。
