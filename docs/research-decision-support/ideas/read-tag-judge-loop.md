# Reading is the only input action: read, tag, judge in one step

Classification and judgment are dropped in passing during reading; there is no separate "filing
phase"; every other view (direction aggregation, provenance graph) is merely a projection of
this one action. The [Rayyan family](../sources/products/rayyan.md) proved the ergonomics of
inline tagging while reading; the
[synthesis matrix convention](../sources/blogs/synthesis-matrix.md) proved that filling in as
you read is the right posture for reviews -- this project merges both onto the source card and
aggregates automatically.
