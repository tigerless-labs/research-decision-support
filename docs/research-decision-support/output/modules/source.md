# source 证据模块

**职责**：参考资料与自身观察的唯一入口；持有全部证据卡，为一切主张提供锚点。

**行为边界**：

- agent 自动收录会话中出现的一切来源，一源一卡。
- 不建主题目录树，类型容器只是薄壳，主题结构由 tag 涌现。
- 入库即分级（综述>论文>文档>帖子，分级表随场景可配）。
- tag 分类在推导连线步进行且**可选**——人工主动打或 agent 建议均可；无 tag 的
  卡合法存在、照常被引用与装配，画板落未分类区待补。
- 只存"材料说了什么"，判断不入此层。
